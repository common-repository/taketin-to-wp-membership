== Description ==
<div>
  TAKETIN Marketing Platformで作成した商品毎に会員サイトを用意することができます。
</div>
<div>
  TAKETIN MPの詳細はこちらからどうぞ
</div>
<a href="https://taketin.com" target="_blank">https://taketin.com</a>

== Installation ==

1. Upload \`taketin-to-wp-membership\` to the \`/wp-content/plugins/\` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
== Frequently Asked Questions ==

**無料で使えますか？**

はい、無料で使えますが、TAKETINシステムを導入の方のみ機能させることが出来ると思います。
